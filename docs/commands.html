cppkg-cli Command Reference

Download C/C++ packages into a shared include directory or project workspace. 

npm install -g cppkg-cli
cppkg-cli --help

Package Setup

cppkg-cli init cppkg-cli add nlohmann/json --install

Project Build

cppkg-cli compile src/main.cpp -o app cppkg-cli build --release

Compiler Env

cppkg-cli compiler list cppkg-cli compiler install clang-18
